
 * 极简通用剪切板 多端同步 (PHP 8.0+ 优化版)
  1. 首次运行设置密码
  2. 扫码免密登录
  3. 长轮询实时同步

核心思路：

自动初始化密码：如果 data/config.json 不存在，第一次访问会提示你“设置初始密码”，之后凭此密码登录。

安全性优化：密码使用 password_hash (Bcrypt) 存储，比 sha1 更安全，符合 PHP 8.5 的安全实践。

扫码逻辑联动：PC 端登录后，二维码包含一个动态 Token，手机扫码即刻登录。

在电脑端登录账户后，手机端扫描二维码后登录账户，在任意一端输入文字，另一端能同时出现这个文字.

PHP 8.5 兼容性：使用了联合类型、match 表达式等现代语法，修复了 PHP 8.x 中对 null 和未定义数组键更严格的报错。

双端同步 Bug 修复：引入了 mtime（毫秒级修改时间）校验，确保“谁先输入谁优先，后输入者自动拉取最新”。

更优雅的实时传递：虽然 WebSocket 是最优雅的，但在简单的 PHP 环境中部署成本极高。使用了 长轮询 (Long Polling) 模式。请求发送到后端后，如果没有新内容，PHP 会挂起连接直到有数据更新或超时。这比单纯的定时请求（Polling）更实时且节省带宽。

扫码自动登录：PC 端登录后，生成的二维码包含了一个带有有效期的临时 token。手机扫码后，后端校验 token 并直接发放 Cookie，实现免密登录。

统一使用 JSON 文件存储，逻辑更清晰。
